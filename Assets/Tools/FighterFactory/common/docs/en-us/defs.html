Menu Definitions

The Definitions menu is used to manipulate the project definitions.

See Definitions workspace for more details.

New

Create a new blank document.

Open

Open an existing document. Note: Only the definitions file will be loaded.

Save

Overwrite the last saved document, if it exist. Else, execute 'Save as...'.

Save as UTF-8

Overwrite the last saved document, if it exist, otherwise, execute 'Save as...', using UTF-8 encoding. This performs like regular Save, but converts current encoding to UTF-8 before saving, so make sure everything is right in the current encoding before using this.

Save as...

Save the document on a new file.
