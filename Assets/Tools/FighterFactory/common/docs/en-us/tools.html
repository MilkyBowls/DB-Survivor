Menu Tools

The Tools menu is used to access aditional options and tools. The contents of this menu can vary, because it's used for scripts and engine specific tools too.

Options

Displays the Options window.

Advanced palette editor

Displays the Advanced palette editor.

Script editor

Displays the Script editor.

M.U.G.E.N. only

Export to HTML

Displays the Export to HTML window if current project supports it.
