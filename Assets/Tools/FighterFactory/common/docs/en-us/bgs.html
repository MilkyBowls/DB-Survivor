Menu Backgrounds

The Backgrounds menu is used to manipulate the backgrounds (BGs) of the project.

BGs are part of the definitions file for Stage, Motif and Storyboard, so it's treated as a virtual file pointing to definitions.

See Backgrounds workspace for more details.

Add

Add one BG, starting on the next position from the current.

The 'Add background...' window will be shown. In this window, you can choose the name and type of the BG

Delete

Removes the current background element. Note: A confirmation will be shown if is defined in the Options window.

Duplicate

Duplicate current background element.

Play

Starts BGs simulation playback. In this mode, fields are disabled, background is clipped to screen size and you can move the camera limited to stage bounds. Use arrow keys to move, Hold Shift to move 2px and Alt to move 10px.

Pause

Pauses BGs simulation playback. You regain control over editing fields.

Stop

Stops playback and resets to default editing mode, without screen constraints.

Controllers

Add definition

Adds a BGCtrlDef after current background controller definition.

Remove definition

Deletes current background controller definition.

Add

Adds a BGCtrl after current background controller.

Remove

Deletes current background controller.
