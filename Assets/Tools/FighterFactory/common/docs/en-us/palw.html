Palettes

The Palettes editor is used to edit the sprite's palettes of current project.

See Palettes menu for more details.

Controls

The Palette editor is placed in the right panel (can vary with the interface preset) of the Sprite editor. All controls available to this editor are listed below:

Palette selection scroll bar

Defines the current palette to work on. When you change the current palette, the current sprite will be viewed with this palette.

Group field

Defines the group number of the current palette.

Index field

Defines the index number of the current palette within its group.

Palette

Displays the current palette.

To select multiple colors you can hold Ctrl to additive selection and Shift to range selection.

To move a color you must hold the slot and move to another position, then, all other colors will be moved to the next position.
