Changes the action number of the player's animation. This controller should be used if you have placed P2 in a custom state via a hit and wish to change P2's animation to one specified in P1's air file. For example, when making throws, use this to change P2 to a being-thrown animation.

Required parameters:

  • value = anim_no (int)
    anim_no is the action number to change to.

Optional parameters:

  • elem = elem_no (int)
    elem_no is the element number within the specified action to start from.

Example:

none
