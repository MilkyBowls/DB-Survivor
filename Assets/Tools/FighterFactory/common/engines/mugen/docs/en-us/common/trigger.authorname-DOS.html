Computes the arctangent (in radians) of the specified argument.

Format: AuthorName [oper] "name"

Arguments:

  • [oper]: =, != (other operators not valid)
  • "name" (string): Name to compare against. Must be in double quotes.

Return type: boolean (0 if false or 1 if true)

Error conditions: none

Example:

trigger1 = Authorname = "Suika"
; Returns true if the character's author is named Suika.
