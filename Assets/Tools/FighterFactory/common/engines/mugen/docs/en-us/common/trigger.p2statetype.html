Gives the opponent's state-type.

Format: P2StateType [oper] state_type

Arguments:

  • [oper]: =, != (other operators not valid)
  • state_type (char): S, C, A. Stand, Crouch and Air state-types.

Return type: boolean (0 if false or 1 if true)

Error conditions: none

Example:

trigger1 = P2StateType = S ; True if the opponent is in a stand-type state.
