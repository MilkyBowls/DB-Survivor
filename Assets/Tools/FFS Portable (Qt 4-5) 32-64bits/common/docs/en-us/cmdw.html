Commands

The Commands editor is used to edit the commands of current project (character only).

See Commands menu for more details.

Controls

All controls available to the commands editor are listed below:

Code block list

Displays a tree list with all code blocks of the document. Some blocks are handled in a special way to display more informations. If you click in the list, the cursor will be placed in the line of the selected item.

Workspace

As all other code based workspaces, the definitions editor have syntax highlight and checking, and follow all code and color options.
