Guide

Fighter Factory Studio is a full featured multi-engine editor. Using it, you can create and edit characters, stages, fonts, and everything you want. This version is a complete rework from Fighter Factory 3. Now, you can use the interface that you want, like Classic and Ultimate versions of Fighter Factory.

Fighter Factory isn't an automated program. It provide all possible tools and functions, allowing the user to use your own creativity and make your creations in a fast and easy way. The program will not do all the work for you.
