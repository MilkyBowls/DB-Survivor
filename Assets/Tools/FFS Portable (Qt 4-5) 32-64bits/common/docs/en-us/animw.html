Animations

The Animations editor is used to edit the animations of current project.

See Animations menu for more details.

Controls

All controls available to the animations editor are listed below:

Animation selection scroll bar

Defines the current animation to work on.

Number

Defines the number of the current animation.

Name

Defines the name of the current animation.

Frame selection scroll bar

Defines the current frame to work on.

Group field

Defines the sprite group of the current frame.

Index field

Defines the sprite index of the current frame.

X axis field

Defines the x axis (offset) of the current frame.

Y axis field

Defines the y axis (offset) number of the current frame.

Time

Defines the time/delay of the current frame. You can use -1 to infinite.

Flip

Defines if the sprite will be inverted (vertically, horizontally or both) or not.

Transparency

Defines the sprite transparency, that can be: None, Addition, Addition 50% or Subtract. The 'Addition' transparency have two parameters.

Source

Opacity of the sprite.

Dest

Darkness of the destination surface.

Start loop in this frame

Defines a loop point in this frame. After the last frame, the animation will go to the next loop point from the current (only if the time is positive).

X Scale

Defines the x scale of the frame's sprite. Note: Only on M.U.G.E.N. 1.1.

Y Scale

Defines the y scale of the frame's sprite. Note: Only on M.U.G.E.N. 1.1.

Angle

Defines the angle of the frame's sprite in degrees. Note: Only on M.U.G.E.N. 1.1.

Interpolate

Defines the interpolation between frames. Note: Only on M.U.G.E.N. 1.1.

Blend: interpolate the blending (transparency).

Offset: interpolate the offset (axis).

Scale: interpolate the scale.

Angle: interpolate the angle.

OnionSkin

If enabled, displays the OnionSkin sprite. When enabled, the OnionSkin scroll bar will be set to the current animation. You can control the OnionSkin appearance in the Options window.

OnionSkin animation scrool bar

Defines the current OnionSkin animation to display.

OnionSkin frame scrool bar

Defines the current OnionSkin frame to display.

OnionSkin find animation

Defines the current OnionSkin animation searching by identifier.

Sprite explorer controls (thumbnails)

All controls available to the Sprite Explorer extra editor are listed below:

Thumbnail group selection

Defines the sprite group to be viewed in the thumbnails list.

Add selected thumbnails to the current animation

Adds all selected thumbnails to the current animation as new frames.

Add all thumbnails to the current animation

Adds all thumbnails to the current animation as new frames.

New animation with the selected thumbnails

Creates a new animation with all selected thumbnails as frames.

New animation with all thumbnails

Creates a new animation with all thumbnails as frames.

Thumbnail list

Displays all sprites of the selected group as thumbnails.

Workspace

The Animation editor is used to create and edit the animations of the project. The graphic workspace contains the rulers, guide lines, OnionSkin image, current sprite image, contact boxes, sprite axis cross, frame axis cross and the extra cross. Some of these can be enabled/disabled in the View menu.

You can change the position of the image by dragging it to the desirable position. The guide lines, contact boxes and the frame axis can be placed in the same way. All other points will move the axis and background image.

The contact boxes are handled in a special way. You can select multiple boxes holding Ctrl or Shift keys, remove all selected boxes pressing the Delete key, move by 1px pressing the arrow keys and cut/copy/paste them.

In the Contact box creation mode, you can add new boxes by simply drawing it on the screen. While you keep in this mode, you can draw many boxes you want, including over another existing one. To exit this mode, right-click on the graphic workspace or disable one of the creation options. When you exit this mode, the last box you drawn will be selected.

The zoom can be changed using the mouse wheel.
