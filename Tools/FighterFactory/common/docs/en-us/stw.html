States

The States editor is used to edit the state machine of current project (character only).

See States menu for more details.

All controls available to the states editor are listed below:

Controls

All controls available to the states editor are listed below:

Text block list

Displays a tree list with all code blocks of the document. Some blocks are handled in a special way to display more informations. If you click in the list, the cursor will be placed in the line of the selected item.

New State controller

Creates a new State controller template.

Edit selected State controller

Edit the selected State controller template.

Delete selected State controller

Deletes the selected State controller template.

State controller list

Displays a list with all available State controller templates. To insert a template you must double-click an item in the list.

Workspace

As all other code based workspaces, the definitions editor have syntax highlight and checking, and follow all code and color options.
