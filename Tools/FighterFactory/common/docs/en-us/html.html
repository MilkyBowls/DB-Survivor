Export to HTML

This function is designed to make project documentation for you. Support may vary based on project type and engine.

It is composed by two columns (options and commands). Commands are automatically generated by engine specific module and you can choose the ones to be exported.

Options

Layout

Choose the layout to use.Information generated by this function will replace some keywords in the layout HTML code. See Keywords.

E-mail

Your e-mail address.

Home-page

Your home-page address.

Title

Some title for acknowledgement or comment.

Text

Text to describe 'Title'.

Horizontal line between commands

Command names in bold


Keywords

M.U.G.E.N.

!ff displayname! Project's display name.

!ff titlename! "Name:" in the current language.

!ff titledate! "Date:" in the current language.

!ff titleversion! "Version:" in the current language.

!ff titleauthor! "Auhtor:" in the current language.

!ff charname! Same as display name.

!ff chardate! 'versiondate' parameter in Definitions.

!ff mugenversion! 'mugenversion' parameter in Definitions.

!ff author! 'author' parameter in Definitions.

!ff titleanim! "Palettes:" in the current language.

!ff titleemail! "E-mail:" in the current language.

!ff titlehomepage! "Home-page:" in the current language.

!ff email! E-mail field value.

!ff homepage! Home-page field value.

!ff titletext! Title field value.

!ff text! Text field value.

!ff bigportrait! Sprite 9000,1.

!ff smallportrait! Sprite 9000,0.

!ff anim*! Animated GIF of standing animation. * can be 1 to 12.

!ff titlecomm! "Commands:" in the current language.

!ff commands! Selected commands using images as buttons.
