Configuring

When you open Fighter Factory at the first time, the default configuration is:

  • Neo interface preset, Dark icons, style and colors.
  • All confirmations and notifications active.
  • Memory caching (fast).
  • Font Courier 10, with the System encoding as default when UED can't do the detection.
  • Left panel, right panel, rulers and contact boxes visible.
  • English language.

Fighter Factory keeps the configuration between sessions, including some editor states.

You can change the configuration in two places, described below.

Menu View

This menu defines the visibility and appearance of some controls. For more information see View...

Menu Tools / Options

The Options window of Fighter Factory 3 contains all customizable settings. For more information see Options...
