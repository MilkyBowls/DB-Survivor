Sounds

The Sounds editor is used to edit the sounds of current project.

See Sounds menu for more details.

Controls

All controls available to the sounds editor are listed below:

Sound selection scroll bar

Defines the current sound to work on.

Group field

Defines the group number of the current sound.

Index field

Defines the index number of the current sound within its group.

Informations

Displays some informations about the current sound:

Channels: number of channels.

Sample rate: number of samples per second.

Bits per sample: size of a single sample.

Duration: the duration in seconds.

Tick count: number of tick at 60Hz.

Workspace

The Sound editor is used to manipulate the sounds of the project. The sound workspace contains the waveform viewer, one per channel and a timeline in ticks at 60Hz.

You can change the position of the view area by dragging it to the desirable position.

The zoom can be changed using the mouse wheel.
