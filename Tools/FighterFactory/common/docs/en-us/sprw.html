Sprites

The Sprites editor is used to edit the sprites (images) of current project.

See Sprites menu for more details.

Controls

All controls available to the sprite editor are listed below:

Sprite selection scroll bar

Defines the current sprite to work on.

Controls (alignment mode)


Group field

Defines the group number of the current sprite.

Index field

Defines the index number of the current sprite within its group.

X axis field

Defines the x axis of the current sprite.

Y axis field

Defines the y axis number of the current sprite.

Decompress on load

Defines if the sprite will be decompressed when loaded (default: disabled). Note: M.U.G.E.N. 1.+ only.

Switch to the sprite palette

If enabled, switch to the sprite palette automatically when changing the sprite selection scroll bar.

Show transparency color

If enabled, force the first color of the palette to fully opaque.

OnionSkin

If enabled, displays the OnionSkin sprite. When enabled, the OnionSkin scroll bar will be set to the current sprite. You can control the OnionSkin appearance in the Options window.

OnionSkin scrool bar

Defines the current OnionSkin sprite to display.

Controls (image editor mode)


Layer blend mode

Defines the blend mode of current layer. Only available for RGBA pixel format.

Layer opacity

Defines the opacity of current layer. Only available for RGBA pixel format.

Layers list

Displays all layers and allows you to hide and select them.

Image editor tools

Defines the current tool.

Canvas move: Moves the canvas position on screen.

Move selection: Moves selected portions of the image.

Rectangle selection: Makes rectangle selections.

Free selection: Changes the selection mask drawing with a pencil.

Magic wand: Changes the selection mask painting large areas like Paint bucket. Left click paints selected pixels and right click paint unselected pixels. Shift enables additive mode and Alt subtraction mode.

Color picker: Picks some color to be the foreground one (left click) or background (right click).

Pen: Paints the foreground color with a pencil.

Eraser: Paints the background color with a pencil.

Remap pen: Paints only the first color of the selected slot with the second color of selected slot. You can set color to current slot by holding shift key and left clicking (first color) ou right clicking (second color), or left clicking a color in the palette for the first color or right click for second color.

Paint bucket: Paints nearest pixels with the same color as the point you clicked, until the end of the flow. Contiguous option controls if it paints by flow or all pixels with the same color.

Foreground color: Defines the foreground color. Left click a color in the palette to choose it as foreground, except while Remap pen is active.

Background color: Defines the background color. Right click a color in the palette to choose it as background, except while Remap pen is active.

Workspace

The Sprite editor is used to manipulate the sprites (images) of the project. The graphic workspace contains the rulers, guide lines, OnionSkin image, current sprite image and the axis cross. Some of these can be enabled/disabled in the View menu.

You can change the position of the image by dragging it to the desirable position. The guide lines can be placed in the same way. All other points will move the axis and background image.

When on image editor mode, the workspace displays the current sprite as a layer, so you can use the available editing tools to do changes in the sprite's image. Selections in this mode are a mask, so you can even paint the selection as in professional tools.

The zoom can be changed using the mouse wheel.
