Returns the name of the player's teammate, if present (if not present, this will never trigger). (the internal name specified in the .DEF file, which may not be the same as the displayed name).

Format: P3Name [oper] name

Arguments:

  • [oper]: =, != (other operators not valid)
  • name (string): Name to compare against.

Return type: boolean (0 if false or 1 if true)

Error conditions: none

Example:

none
