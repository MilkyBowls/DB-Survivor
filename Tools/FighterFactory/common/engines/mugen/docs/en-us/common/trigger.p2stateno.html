Gives the opponent's state number.

Format: P2StateNo [oper] value

Arguments:

  • [oper]: =, !=, <, >, <=, >=
  • value (int): Numerical value to compare with.

Return type: boolean (0 if false or 1 if true)

Error conditions: none

Example:

trigger1 = P2StateNo = 200 ; True if the opponent is in state number 200.
